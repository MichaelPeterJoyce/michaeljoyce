"Learn new skills online through our extensive library of online courses, we take pride in optimising personal development and improving employee productivity"

Skillsboost was the first small application I worked on. It was a frontend application that sold Online eLearning courses via PayPal. The frontend was built in Backbone.js connecting to a RESTFul api backend built in PHP Zend 2 Framework.

The application allowed users to browse a store for courses, purchase a course via PayPal and then login to an Account Page where they could track their progress and continue their purchased courses.

  • Backbone.js
  • Underscore.js
  • Bootstrap 3 CSS
  • jQuery
